About me

Thank you for visiting my portfolio website. When I was 16 years old (2005) I built my first website in Notepad with HTML for an acquaintance who was running a massage studio. I built way more websites after that for friends and family, first with HTML, and later with WordPress. But I did not start to make a career out of that. I started my career as a biomedical lab technician. Early on in my career, I found out that I could do my work more efficiently by using programming for analysis. It started with recording simple macros in Excel, then I edited these macros with Visual Basic for more functionality. When the datasets started to become bigger I started using R and when I wanted my colleagues to use my programs who did not have programming experience I started to combine R with Java User Interfaces.

At some point, I realized that the biomedical industry would not make me happy and I decided that I wanted to build a career in web development. So I started a career path course toward full stack developer. Understanding the full stack makes me enthusiastic because I have always been very curious about how all IT systems work together and as a lab technician I was also trying to connect different systems to improve my efficiency.
